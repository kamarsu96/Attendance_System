# Firebase Setup Instructions

To make Firebase Admin work, follow these steps:
1. Go to your Firebase Console (console.firebase.google.com).
2. Go to Project settings > Service accounts.
3. Click "Generate new private key".
4. Save the downloaded JSON file as `firebase-service-account.json` in this same folder (the root of your backend project).

**DO NOT** commit the `firebase-service-account.json` file to version control.
